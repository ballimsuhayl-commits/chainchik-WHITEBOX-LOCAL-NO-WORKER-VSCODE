# VS Code one-click run (Windows)

## 1) Open the folder
VS Code → File → Open Folder → select this project folder.

## 2) Make sure Docker Desktop is running
Open Docker Desktop and wait until it says it is running.

## 3) Run using VS Code Tasks (no typing)
Press **Ctrl + Shift + P** → type **Tasks: Run Task** → pick:

- **WhiteBox: Start (Simple)** (first)
- **WhiteBox: Open Store**
- **WhiteBox: Open Admin**

To stop:
- **WhiteBox: Stop (Simple)**

To check status:
- **WhiteBox: Status (Simple)**

To watch logs:
- **WhiteBox: Logs (Web)** or **WhiteBox: Logs (API)**
